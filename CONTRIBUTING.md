# Contributing to @centy-io/assets

Thank you for your interest in contributing!

## Adding New Assets

1. Add your asset file (SVG, PNG, etc.) to the appropriate directory
2. Update README.md to document the new asset
3. Run `pnpm format` to ensure proper formatting
4. Submit a pull request

## Development Commands

- `pnpm format` - Format files
- `pnpm format:check` - Check formatting
- `pnpm spell` - Check spelling
- `pnpm lint` - Lint package.json

## Commit Messages

Follow [Conventional Commits](https://www.conventionalcommits.org/):

```
feat(logos): add new logo variant
docs(readme): update asset documentation
```
